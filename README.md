# Templates List

## Elsevier

1. **elsevier-cas-double_column**: A double-column template for Elsevier journals.
2. **elsevier-cas-single_column**: A single-column template for Elsevier journals.
3. **elsevier-cas-single_column-numbered_reference**: A single-column template with numbered references for Elsevier journals, derived from the single-column template.
4. **bib_style**: Bibliography style files for Elsevier journals; use with the Elsevier templates above.

## IEEE

5. **IEEE-Transactions-LaTeX2e-templates-and-instructions**: LaTeX template for IEEE journals and transactions (IEEEtran.cls). Includes sample article and how-to guide. Suitable for submission to IEEE periodicals.

## Springer Nature

6. **nature-sn-article-template**: Article template for Springer Nature journals (sn-jnl.cls). Supports multiple reference styles (e.g. Nature, Basic, Vancouver, APA, Chicago, Math & Physical Sciences). Suitable for Nature and other Springer Nature journals.

## Response to Reviewers

7. **response-to-reviewer**: A general LaTeX template for replying to reviewer comments. Provides formatted structure, color-coded sections (reviewer comments / author response / related changes), and bibliography support. See the folder’s `README_Template.md` for usage.

# 模板列表

## Elsevier（爱思唯尔）期刊模板

1. **elsevier-cas-double_column**：适用于 Elsevier 期刊的双栏模板。
2. **elsevier-cas-single_column**：适用于 Elsevier 期刊的单栏模板。
3. **elsevier-cas-single_column-numbered_reference**：基于单栏模板、带编号参考文献的 Elsevier 期刊模板。
4. **bib_style**：Elsevier 期刊参考文献样式文件，请与上述模板配合使用。

## IEEE 期刊模板

5. **IEEE-Transactions-LaTeX2e-templates-and-instructions**：适用于 IEEE 期刊与汇刊的 LaTeX 模板（IEEEtran.cls），含示例文章与使用说明。

## Springer Nature 期刊模板

6. **nature-sn-article-template**：Springer Nature 期刊文章模板（sn-jnl.cls），支持多种参考文献样式（如 Nature、Basic、Vancouver、APA、Chicago、数理科学等），适用于 Nature 及 Springer Nature 旗下期刊。

## 回复审稿意见模板

7. **response-to-reviewer**：通用回复审稿意见 LaTeX 模板，提供格式化结构、颜色区分（审稿人意见 / 作者回复 / 修改说明）及参考文献支持。详细用法见该目录下的 `README_Template.md`。

---

# How to use the templates

1. Set up a LaTeX environment (local install or online editor such as Overleaf).
2. Copy the template you need into your project.
3. Replace placeholder text with your own content.
4. Compile the document to produce the output.

# 如何使用这些模板

1. 确保 LaTeX 编译环境已配置好（本地安装或使用 Overleaf 等在线编辑器）。
2. 将需要使用的模板复制到你的项目中。
3. 用你自己的内容替换模板中的占位符文字。
4. 编译文档，查看最终效果。
