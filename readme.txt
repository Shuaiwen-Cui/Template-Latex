# Templates List

Elsevier:

1. elsevier-cas-double_column: A double-column template for Elsevier journals.
2. elsevier-cas-single_column: A single-column template for Elsevier journals.
3. elsevier-cas-single_column-numbered_reference: A single-column template with numbered references for Elsevier journals, derived from the single-column template.
4. bib_style: bibiography style files for Elsevier journals, please use with the Elsevier templates.

# 模板列表

**Elsevier（爱思唯尔）期刊模板：**

1. **elsevier-cas-double_column**：适用于Elsevier期刊的双栏模板。  
2. **elsevier-cas-single_column**：适用于Elsevier期刊的单栏模板。  
3. **elsevier-cas-single_column-numbered_reference**：基于单栏模板，带有编号参考文献的Elsevier期刊模板。  
4. **bib_style**：Elsevier期刊使用的参考文献样式文件，请与上述模板配合使用。

# How to use the templates:

1. Ensure your Latex environment is set up, either locally or using an online editor like Overleaf.
2. Copy the template you want to use into your project.
3. Replace the placeholder text with your own content.
4. Compile the document to see the output.

# 如何使用这些模板：

1. 确保你的 LaTeX 编译环境已设置好，可以在本地安装，或使用在线编辑器如 Overleaf。  
2. 将你想使用的模板复制到你的项目中。  
3. 用你自己的内容替换模板中的占位符文字。  
4. 编译文档，查看最终效果。
